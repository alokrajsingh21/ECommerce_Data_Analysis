# ECommerce_Data_Analysis
This project performs Exploratory Data Analysis (EDA) on the Superstore USA e-commerce dataset to uncover meaningful business insights related to sales, profit, customer behavior, shipping modes, and product performance.

Project Overview
1.Analyzed 9,426 records with 24+ features including orders, customers, products, regions, sales, and profit.
2.Focused on understanding trends across order priority, shipping modes, customer segments, product categories, and time (year-wise analysis).
3.Cleaned and preprocessed data by handling missing values, correcting inconsistent categories, and creating derived features (e.g., Order Year).

Tools & Technologies
1.Python
2.Pandas & NumPy – Data cleaning and manipulation
3.Matplotlib & Seaborn – Data visualization
4.Google Colab
